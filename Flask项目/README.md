# 简易图书管理系统

## 准备工作

1. 在login.py中，将server变量设置为你的数据库服务器地址

login.py

```python
server = "" # 改成你的数据库服务器
```

2. 按照LibraryManageSystem.sql搭建数据库LibraryDB
3. 下载python对应的库  
python=3.12.3

```bash
pip install -r requirements.txt
```

## 运行

运行app.py,之后打开[http://127.0.0.1:5000](http://127.0.0.1:5000)进入系统。
